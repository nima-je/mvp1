# LinkShop Theme

قالب فروشگاهی تک‌فروشنده برای ووکامرس با تمرکز بر تجربه کاربری فارسی و راست‌چین.

## نصب
1. مخزن را در مسیر `wp-content/themes/linkshop` کلون کنید.
2. در پیشخوان وردپرس، قالب LinkShop را فعال کنید.
3. ووکامرس را نصب و تنظیمات پایه فروشگاه را انجام دهید.
4. از سفارشی‌ساز برای بارگذاری لوگو و انتخاب رنگ اصلی استفاده کنید.

## امکانات کلیدی
- طراحی مدرن با گلس مورفیسم
- پشتیبانی کامل ووکامرس (صفحات فروشگاه، محصول، سبد، پرداخت، حساب کاربری)
- سفارشی‌سازی محدود: لوگو، عنوان سایت، رنگ اصلی
- ماژول پیامک (ثبت در لاگ برای توسعه بعدی)
- سیستم لایسنس دامنه‌محور (هشدار در صورت نامعتبر بودن)
- داشبورد آنالیتیکس داخلی و ثبت شاخص‌های روزانه
- لینک فاکتور HTML برای هر سفارش
